

    -< Weevil >-


    Available Commands:
    
    1. play
        - Start playback of a track or playlist via its yt-URL.
        - Flags:
            --url, -u: Specify the URL of the music.
            --output, -o: Specify the output folder for downloaded music. (default: cwd)
            --fileType, -f: Specify the file type (default: any).

    2. pause, p
        - Pause the current playback.

    3. resume, r
        - Resume playback from a paused state.

    4. exit, close, quit
        - Exit Weevil.

    5. get
        - Get information about various configuration options.
        - Flags:
            --key, -k: Specify the configuration key.

            --title_declarations, -t: Get information about title declarations.
            --info, -i: Get information about casual information settings. (default: true)
            --failures, -f: Get information about failure settings.
            --warnings, -w: Get information about warning settings.

    6. set
        - Set various configuration options.
        - Flags:
            --key, -k: Specify the configuration key.
            --value, -v: Specify the configuration value.

            --title_declarations, -t: Set title declarations.
            --info, -i: Set casual information display. (default: true)
            --failures, -f: Set failure display.
            --warnings, -w: Set warning display.
            --volume, -vol: Set the volume (TODO).

    7. skip, s
        - Skip to the next track in the playlist.

    8. list_playlists, lp
        - List available playlists.
        - Flags:
            --directory, -dir: Specify the directory of playlists (default: cwd).
            --show_id, -si: Show playlist IDs (default: true).

    9. help, -h, --help
        - Display this help message.

    10. clear, cls
        - Clear the terminal

    Example Usage:
        $ play --url="<music_url>" --output="<output_folder>" --fileType="mp4"
        $ pause
        $ set -i="true"
        $ lp --show_id="true"
